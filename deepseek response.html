Azure Native Services

  3. Key Features:

  4. Use Cases: IoT telemetry, real-time dashboards, fraud detection.
  5. Pros: Low operational overhead, serverless, easy to set up.

  6. Cons: Limited to SQL for complex logic; less flexibility for custom code.

  7. Azure Event Hubs + Azure Functions

  8. Description: Event-driven architecture using Event Hubs for ingestion and Functions for lightweight processing.
  9. Key Features:
    • Scalable event ingestion (millions of events/sec).

  10. Use Cases: Simple event filtering, alerting, or triggering workflows.
  11. Pros: Cost-effective for small-scale use cases; fully managed.

  12. Cons: Limited state management; not suited for complex aggregations.

  13. Azure Data Explorer (Kusto)

  25. Azure Synapse Analytics (Near-Real-Time)
